# Mirage Chat App

Mirage stands for **Mildly Interesting, Really Average, Generally Entertaining**. It's a simple chat application designed for casual conversations.

## Usage Instructions

1. **Downloading and Running the Client**
   - Clone the repository to your local machine.
   - Ensure you have Python installed.
   - Install the necessary dependencies using `pip install -r requirements.txt`.
   - Run the client application using `python client.py`.

2. **Connecting to the Server**
   - By default, the client connects to `http://127.0.0.1:5000` for server operations.
   - Modify the `server_url` variable in `client.py` to change the server URL if hosting on platforms like PythonAnywhere or Render.com.

3. **Using Mirage**
   - Select a room or create a new one.
   - Send messages by typing in the text entry and pressing 'Send'.
   - Leave a room using the 'Leave Room' button.

4. **Hosting on External Platforms**
   - If hosting the server on platforms like PythonAnywhere or Render.com, update the `server_url` in `client.py` to the appropriate URL provided by the hosting service.

5. **Privacy Note**
   - Mirage does not store your messages on the server permanently. Messages are stored temporarily for a brief period (1.9 seconds) to facilitate message delivery.

## Disclaimer
Mirage is intended for non-sensitive, casual communications. It does not provide encryption or long-term message storage. For secure communications, consider using dedicated messaging services.
